Desenvolvendo o Jogo Torre de Hanói

Nesta página, vamos entender como desenvolver o jogo Torre de Hanói com explicações bem simples. Se você está começando agora, não se preocupe, vamos ver tudo de forma clara e fácil!

O Básico do Código

Primeiro, vamos dar uma olhada no código que vamos usar para o jogo. Cada parte será explicada com calma.

                    
// Função recursiva para resolver Torre de Hanói
function torreDeHanoi(n, origem, destino, auxiliar) {
    if (n === 1) {
        console.log(`Mover disco 1 de ${origem} para ${destino}`);
        return;
    }
    torreDeHanoi(n - 1, origem, auxiliar, destino);
    console.log(`Mover disco ${n} de ${origem} para ${destino}`);
    torreDeHanoi(n - 1, auxiliar, destino, origem);
}

// Chamando a função para 3 discos
torreDeHanoi(3, 'A', 'C', 'B');

O que é uma Função?

Uma função é como uma receita de bolo! Ela tem um nome, e quando você pede para fazer essa "receita", ela executa as instruções que foram escritas dentro dela.

No nosso código, a função se chama torreDeHanoi, e ela é chamada para resolver o jogo. Quando pedimos para essa função rodar, ela começa a trabalhar e fazer as movimentações necessárias.

O que os Parâmetros Fazem?

Os "parâmetros" são como os ingredientes de uma receita. Eles são informações que a função precisa para funcionar direito.

  • n: Quantos discos temos no jogo (pense como se fossem camadas de um bolo).
  • origem: De onde os discos começam.
  • destino: Para onde queremos levar os discos.
  • auxiliar: Uma torre extra que ajuda a movimentar os discos.

Então, quando chamamos a função torreDeHanoi(3, 'A', 'C', 'B'), estamos dizendo: "Mova 3 discos da torre A para a torre C, usando a torre B para ajudar".

O que é Recursão?

Recursão é uma palavra chique, mas o conceito é bem simples. Pense assim: é como uma instrução que se repete várias vezes até que o trabalho termine.

Quando a função torreDeHanoi é chamada, ela continua chamando a si mesma para resolver partes menores do problema. Como se fosse uma pessoa resolvendo um problema aos poucos, uma parte de cada vez.

Entendendo o Código em Ação

Quando você pede para o código rodar, ele vai mostrar no console os movimentos que os discos precisam fazer, assim:

                    
Mover disco 1 de A para C
Mover disco 2 de A para B
Mover disco 1 de C para B
Mover disco 3 de A para C
Mover disco 1 de B para A
Mover disco 2 de B para C
Mover disco 1 de A para C

Esses são os movimentos necessários para mover os discos de uma torre para outra, resolvendo o jogo!

Brincando com o Código

Agora que você entendeu o básico, pode tentar mudar o número de discos. Experimente trocar o número 3 no código torreDeHanoi(3, 'A', 'C', 'B') por outro número. Isso vai mudar a quantidade de movimentos necessários para completar o jogo.

Tente também alterar as torres de destino e auxiliar para ver como o jogo se adapta.

Aprenda como funciona o Ghost Atack e a sua lógica por trás

Resumo da Lógica do Jogo

  • Início do Jogo: O jogador escolhe um nível e clica em "iniciar", iniciando o jogo.
  • Exibição de Fantasmas: Fantasmas aparecem em posições aleatórias na tela, e o jogador deve clicar neles para ganhar pontos.
  • Vidas: O jogador começa com 3 vidas. Se um fantasma não for clicado antes de desaparecer, o jogador perde uma vida.
  • Contagem Regressiva: O jogador tem 20 segundos para clicar nos fantasmas.
  • Fim do Jogo: O jogo termina se o tempo acabar ou se o jogador perder todas as vidas. O jogador pode então optar por reiniciar o jogo.

    • 1. Seleção de Elementos HTML:

  • O código começa selecionando elementos da página HTML que têm IDs ou classes específicas. Esses elementos são usados para iniciar o jogo, mostrar as telas do jogo, e manipular o tempo e as vidas.

    • 2. Inicialização do Jogo:

    Aqui, o código aguarda um clique no botão de "início" (btnStart). Quando o botão é clicado:

  • Ele define o nível de dificuldade com base no valor selecionado pelo jogador.
  • Ele exibe corações cheios (representando vidas) na tela.
  • A página inicial é escondida e a página do jogo é exibida.
  • As funções startInteval(levelGame) e startTime() são chamadas para começar o jogo.

    • 3. Variáveis e Controles do Jogo:

    O código declara variáveis para largura e altura da janela, vidas do jogador (iniciando em 1), o intervalo do jogo, e o tempo disponível para o jogo (20 segundos). O tempo é exibido na tela.

    4. Funções para Tamanho da Página e Posições Aleatórias:

  • sizePage: Atualiza a largura e altura da janela sempre que a janela é redimensionada.
  • randomSize: Retorna uma classe aleatória para os fantasmas (diferentes estilos).
  • randomPosition: Gera uma posição aleatória para os fantasmas aparecerem na tela.

    • 5. Lógica do Fantasma:

    Se um fantasma já estiver presente, ele é removido. Se o jogador já tiver perdido 3 vidas, o jogo termina, e a tela de "Game Over" é mostrada. Se ainda houver vidas, uma vida é removida.

    6. Interações e Contagem do Tempo:

    O jogador pode clicar no fantasma para removê-lo, e a função startTime começa uma contagem regressiva de 20 segundos, atualizando o tempo na tela a cada segundo. Quando o tempo chega a 0, o jogo termina.

    7. Parar o Jogo:

    A função stopGame verifica se o tempo acabou. Se sim, ela para os intervalos do jogo e mostra a tela de vitória.

    8.Reiniciar o Jogo:

    Os botões de reiniciar e jogar novamente recarregam a página, reiniciando o jogo.
